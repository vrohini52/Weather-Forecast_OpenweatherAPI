# Weather Data Science Application

## Project Overview

This project is a Python-based weather information application that retrieves real-time weather data using the OpenWeatherMap API. The application allows users to search for weather conditions in different cities, store search history, and visualize temperature data.

The system demonstrates the integration of APIs, data processing using Pandas, and data visualization using Matplotlib.

---

## Features

* Retrieve real-time weather data using OpenWeatherMap API
* Menu-driven user interface
* Store weather search history using CSV
* Display past searches
* Visualize temperature data using Matplotlib
* Error handling for invalid city inputs

---

## Technologies Used

* Python
* Requests Library (API calls)
* Pandas (Data storage and manipulation)
* Matplotlib (Data visualization)
* OpenWeatherMap API
